# agromarket-catalogo
Módulo encargado de mostrar la lista de productos agrícolas disponibles en la plataforma.
Permite consultar precios, detalles y contactar a los productores.
Incluye filtros por categorías para facilitar la búsqueda de los usuarios.
Forma parte del sistema principal de comercialización AgroMarket.
